📋 NOVO FORMATO CSV (6 COLUNAS)

  Fabricante;Descrição;Preço Distribuidor s/ IPI;Preço Distribuidor c/ IPI;Preço Unit. Final s/ IPI;Preço Final c/ IPI

  Exemplo:
  5L500;BATERIA DE LITIO NAO RECARREGAVEL;2.288,58;2.511,72;2.542,87;2.790,80

  ✓ Separador entre colunas: ponto-e-vírgula (;)
  ✓ Separador de decimal: vírgula (,)
  ✓ Sem aspas
  ✓ Sem linhas vazias
  ✓ Sem códigos duplicados

🗄️ AS 6 COLUNAS DO CSV

  COLUNA 1: Fabricante
    → Código único do produto (ex: 5L500)

  COLUNA 2: Descrição
    → Nome do produto (ex: BATERIA DE LITIO)

  COLUNA 3: Preço Distribuidor s/ IPI
    → Preço sem imposto para distribuidor (ex: 2.288,58)

  COLUNA 4: Preço Distribuidor c/ IPI
    → Preço com imposto para distribuidor (ex: 2.511,72)

  COLUNA 5: Preço Unit. Final s/ IPI
    → Preço final sem imposto (ex: 2.542,87)

  COLUNA 6: Preço Final c/ IPI
    → Preço final com imposto (ex: 2.790,80)

⚠️ IMPORTANTE

  O comando DROP TABLE (no SQL) DELETA TODOS OS DADOS sem recuperação!

  SE QUER MANTER DADOS ANTIGOS:
    → Remova a linha: DROP TABLE IF EXISTS products CASCADE;
    → Depois execute o resto do SQL

  SE QUER DELETAR TUDO E COMEÇAR DO ZERO:
    → Execute o SQL completo (com DROP TABLE)
