================================================================================
                        📋 NOVO FORMATO CSV
================================================================================

Seu arquivo CSV deve ter EXATAMENTE 6 colunas:

Fabricante;Descrição;Preço Distribuidor s/ IPI;Preço Distribuidor c/ IPI;Preço Unit. Final s/ IPI;Preço Final c/ IPI

Exemplo:
5L500;BATERIA DE LITIO NAO RECARREGAVEL;2.288,58;2.511,72;2.542,87;2.790,80

REGRAS IMPORTANTES:
✓ Use ponto-e-vírgula (;) para separar colunas
✓ Use vírgula (,) para decimais (2.288,58 SIM | 2288.58 NÃO)
✓ Não use aspas ao redor de valores
✓ Sem espaços extras
✓ Sem linhas vazias
✓ Sem códigos duplicados

================================================================================
                         ⚠️ IMPORTANTE
================================================================================

O comando DROP TABLE (no SQL) DELETA TODOS OS DADOS ANTIGOS sem possibilidade
de recuperação.

SE VOCÊ QUER MANTER DADOS ANTIGOS:
→ Abra EXECUTE_ESTE_SQL.md
→ Remova a linha: DROP TABLE IF EXISTS products CASCADE;
→ Execute o resto do SQL

SE VOCÊ QUER DELETAR TUDO E COMEÇAR DO ZERO:
→ Execute o SQL completo (com DROP TABLE)
